### Desarollo ChatBot
#### Configuración de la API Key
Este chatbot usa la API de Gemini. Para ejecutarlo se necesita una clave propia:

Ingresa a Google AI Studio.
Genera una API Key.
Abre el archivo `CHATBOT.py`.
Reemplaza el texto `"API_KEY_GEMINI"` en la variable `API_KEY` por tu clave.
